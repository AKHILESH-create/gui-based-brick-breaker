# 🧱 Brick Breaker Game 

## 🧱 Overview

The **Brick Breaker Game** is a GUI-based arcade game developed using **Java Swing**, **AWT**, and **Object-Oriented Programming (OOP)** concepts. The game challenges players to break all bricks using a bouncing ball controlled by a movable paddle.

This project focuses on real-time game mechanics, collision detection, keyboard interaction, score tracking, and level-based gameplay implementation.

## ✨ Features

* 🎮 Interactive paddle and ball movement
* 🧱 Brick collision and destruction system
* 📈 Real-time score tracking
* 🏆 Multiple game levels with increasing difficulty

## ⚙️ How the Game Works

1. The player controls the paddle using keyboard keys.
2. The ball starts bouncing automatically.
3. The ball destroys bricks on collision.
4. Each brick destroyed increases the score.
5. After clearing all bricks, the player moves to the next level.
6. The ball speed increases with each level.
7. The game ends if the ball touches the bottom boundary.
